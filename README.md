# NLayerSampleCode

نمونه کد سی شارپ Windows Form Application برای معماری Nlayer و استفاده از ADO.NET

در این نمونه کد سعی شده است قواعد Clean Code رعایت شود.
لایه ها وظایف مشخص دارند و این وظایف تفکیک شده است .
کدها و وظایف بصورت نمونه و خلاصه نوشته شده است که مناسب یادگیری و الگو گیری باشد.
شبیه سازی فرم لاگین در کدها وجود دارد و پیاده سازی دپشده است.
ذخیره آخرین کاربر لاگین شده در رجیستری ویندوز و واکشی آن در کدها وجود دارد.
ارث بری فرم ها از از فرم والد وجود دارد.
از کامپوننت تلریک هم استفاده شده است که dll های آن در کنار سورس موجود است 
دیتابیس SQL SERVER می باشد که در کنار سورس قرار گرفته است.
برای فراخوانی فرم ها از پترن singelton استفاده شده است/
از Binding برای اتصال اجزای فرم به دیتامدل ها استفاده شده است و نحوه استفاده از بایندینگ در ویندوز فرم را میتوانید بصورت عملی یاد بگیرید.

در پیاده سازی ها برخی ریزه کاری ها و کدهایی وجود دارد که میتوانید از آن در پروژه های خود استفاده کنید.
